如何解决Facebook 无法点击See More 查看更多

发表于 2021-01-15 阅读次数：

相信很多人最近在更新了最新的iOS 或 iPadOS 和最新的 Facebook 后，都会发现某些评论的 See More / 查看更多 点击后无反应，需要手动拷贝粘贴才可以看到完整评论.

如今我们可以通过某些手段来解决这个Bug.

步骤

  1. 打开 Facebook , 点击 Menu.
  2. 向下滑找到 Settings & Privacy , 点击里面的 Settings.
  3. 进入选项后，下滑找到 Media and Contacts , 找到 Browser 并点击进去.
  4. 点击 Clear Data.
  5. 从后台直接关闭 Facebook 并且再次打开，理论上现在可以点击 See More / 查看更多了！
