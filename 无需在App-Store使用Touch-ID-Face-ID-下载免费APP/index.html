无需在App Store下载免费APP时验证Touch ID / Face ID

相信各位都会有App Store 下载免费APP的时候都需要Touch ID / Face ID 的困扰吧.
今天，我们将会教大家如何无需在App Store下载免费APP时验证Touch ID / Face ID.

步骤

  1. 点击进入Settings
  2. 在 Settings里面，下拉找到 Touch ID / 面容ID与密码 .
  3. 点击并且输入密码.
  4. 输入密码后，找到 将面容/Touch ID用于iTunes 与 App Store 一栏，并关掉该选项.
  5. 退出面容ID/Touch ID与密码，返回Settings 的主页面.
    接下来，向上滑，找到屏幕使用时间 , 点击进入.
  6. 点进去了后，在使用时间数据表的下方，找到内容与隐私纺问限制，点击进去.
  7. 点进去内容与隐私纺问限制后，找到iTunes 与App Store购买项目 , 点击进去.
  8. 找到需要密码一栏，选择 不需要 ，即可解决.

总结

以上的教程确实可以在你下载App Store内的免费APP时不会要求 Touch ID / 面容ID / 密码 的严正要求，不过 付费APP还是需要验证哟！
