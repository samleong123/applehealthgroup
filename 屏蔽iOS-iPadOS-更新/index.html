屏蔽iOS / iPadOS 更新

发表于 2020-12-23 阅读次数：

相信大多数人手上的旧机型iPhone 应该都不想升级最新的系统，毕竟升级了后可以明显地感受到延迟和耗电量增加. 因此，我们需要通过配置文件来屏蔽更新.
这里我们使用编辑手上的 iPhone 6s - iOS 12.4.1 作为演示.

还未安装屏蔽配置文件前，可以看到，点入软件升级是会提示升级iOS 14.3

好了，接下来开始教程！

屏蔽iOS / iPadOS 更新

请满足以下条件

  • 网络连接
  • 可安装配置文件

步骤

  1. 请使用Safari浏览此网页，然后点击屏蔽iOS / iPadOS 更新配置文件 来下载配置文件.

    点击 Allow / 允许 下载该配置文件，接着点击 Close / 关闭

  2. 依次打开 Settings/设置 -> 已下载的描述文件 / Profile Downloaded -> 点击右上角的 Install / 安装

    打开 Settings/设置

    打开 已下载的描述文件 / Profile Downloaded

    点击右上角的 Install / 安装

  3. 依次 输入密码 -> 点击右上角 Done/完成

    输入密码 -> 点击右上角 Done/完成

  4. 点击右上角的 Install / 安装

    点击右上角的 Install / 安装

  5. 再次点击 Install / 安装

    再次点击 Install / 安装

  6. 接下来，他会要求重启手机，点击 Restart / 重启

    点击 Restart / 重启

  7. 重启后,依次打开 Settings/设置 -> 通用 / General -> 软件更新 / Software Update

    是不是发现没更新了呢？

到了第7步，就代表成功了哦！

至此，你的旧iOS机型/iPadOS机型就能停留在这个iOS版本了！除非手动去除此配置文件/使用iTunes/爱思助手等等之类的工具更新iOS设备/iPadOS设备,否则是可以永久屏蔽更新的！
